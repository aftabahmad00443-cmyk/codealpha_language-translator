#  Language Translation tool

Interactive language translator using **Google Translate** and **text-to-speech** with **Gradio**.  

This app allows you to:

- Translate text between multiple languages.
- Hear the translated text spoken aloud.
- Use a clean, interactive interface in your browser.

---

## **Features**

- Text input for translation.
- Source and target language dropdowns.
- Safe text-to-speech (TTS) support for multiple languages.
- Works locally or can be deployed online.

---
